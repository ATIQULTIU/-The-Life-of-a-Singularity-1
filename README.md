## ✨ Overview

**The Life of a Singularity** is an interactive WebGL visualization that simulates the lifecycle of a black hole. Using **Three.js**, **GLSL shaders**, and **GSAP**, the project creates a realistic accretion disk, gravitational lensing effects, dynamic orbital motion, and cinematic camera transitions.

This project demonstrates modern browser-based graphics programming and shader techniques for creating visually immersive space simulations.
